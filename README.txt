Este projeto desenvolvido para uma aplicação web desenvolvida com  os tópicos aplicados em sala de aula:
HTML
CSS
Flask
Banco de Dados

Imagem usada foi inspirada na minha filha, usando Gemini para criar uma imagem coerente 
ao projeto desenvolvido.

Com objetivo de agendar os horários de medicação de crianças, ainda falta
implementar muitas informações, mas já atende alguns requisitos exigidos para a avaliação.

A aplicação permite cadastrar informações como:
- Nome da criança
- Nome do medicamento
- Tipo de dosagem
- Quantidade

Os dados são armazenados em um banco de dados local.
